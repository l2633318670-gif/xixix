## 1. 网站是怎么运作的

- 网站代码放在 **GitHub** 仓库里（`l2633318670-gif/xixix`）。
- 用 **Vercel** 连接 GitHub，**自动部署**：只要 `main` 分支有更新，几分钟后线上 https://machlinkchina.cc.cd 就会自动更新。
- 所以：**改代码 → 合并进 main → 网站自动上线**。你不需要手动上传文件。
- 改动建议先在分支上做，再通过 PR 合并到 main（最稳妥，可回溯）。

> 💡 不懂代码也没关系：要改什么，直接告诉 Claude（我），描述清楚需求即可。

## 2. 怎么看访客数据（Google Analytics）

**GA 看的是：有多少人来、来自哪里、看了什么、有没有点 WhatsApp。**

### 手机查看
1. 打开 https://analytics.google.com 或装「Google Analytics」App，登录你的 Google 账号
2. 选「二手机床 / machlink.cn」这个媒体资源
3. 重点看这几个报告：
   - **实时（Realtime）**：现在有几个人在线、在看哪页（验证统计是否生效就看这个）
   - **报告 → 获取客户 / 流量获取**：访客从哪来（Google 搜索 / 直接 / WhatsApp）
   - **报告 → 用户属性 / 地理位置**：访客来自哪些国家（尼日利亚、UAE 等）
   - **互动 → 事件**：能看到 `whatsapp_click`（点了 WhatsApp）、`generate_lead`（潜在客户）、`email_click`

### ⭐ 必做一次：把"咨询"标记为转化
1. GA 后台 → **管理（左下齿轮）→ 事件（Events）**
2. 找到 **`generate_lead`** 这一行 → 打开右边「**标记为关键事件 / 转化**」的开关
3. 之后在报告里就能直接看到**有多少人发起了咨询**，以及哪个页面/机器带来的最多

> ⚠️ 事件第一次出现，要等有真实点击后约 24 小时才会显示在列表里。

## 3. 怎么看搜索表现（Search Console）

**GSC 看的是：别人在 Google 搜什么词找到你、排第几、被收录了没。**

1. 打开 https://search.google.com/search-console ，登录同一个 Google 账号
2. 选资源 `machlinkchina.cc.cd`
3. 重点看：
   - **效果（Performance）**：被搜的关键词、展示次数、点击次数、平均排名
   - **网页编入索引 / 网页（Pages）**：哪些页面被 Google 收录了
   - **站点地图（Sitemaps）**：`sitemap.xml` 的状态（成功 = 正常）

### 想让新页面更快被收录
- 顶部「**网址检查**」框 → 粘贴新页面网址 → 回车 → 点「**请求编入索引**」
- 每加一台新机器，建议把它的详情页网址这样请求一次。

## 4. 怎么加一台新机器

### 最简单的方式（推荐）
直接把这些发给 Claude（我）：
- 机器**型号 / 名称**（如 "X5040 Knee Mill"）
- **照片**（1 张或多张）
- **类型**：Turret（炮塔）/ Knee（升降台）/ Drill（钻铣）
- 已知的**规格参数**（有多少给多少，没有我用该型号典型值并标注"以视频为准"）
- 一句**状况描述**（如 "成色好，带 DRO"）

我会自动完成下面所有步骤并上线。

### 技术上涉及哪些文件（供了解）
加一台机器需要改 4 处：
1. `images/products/` 里放机器照片
2. `machines/` 里新建一个详情页 HTML
3. `products.html` 里加一张产品卡片（链接到详情页）
4. `sitemap.xml` 里加这台机器的网址

> 这些手动做容易出错，**建议交给 Claude**。

## 6. 网站文件结构速查

```
xixix/
├── index.html              首页
├── products.html           机器列表页
├── about.html              关于我们
├── contact.html            联系页（含询盘表单）
├── news.html               新闻/指南索引
├── 404.html                找不到页面时显示
├── analytics.js            Google 统计 + WhatsApp 点击追踪（GA ID 在这）
├── robots.txt              告诉搜索引擎/AI爬虫规则
├── sitemap.xml             所有页面清单（给 Google）
├── llms.txt                给 AI 引擎（ChatGPT等）看的网站说明
├── machines/               6 个机器详情页
├── news/                   3 篇买家指南文章
├── images/                 机器照片
│   ├── products/           列表/详情页用的机器图
│   └── og/                 社交分享卡（分享到 WhatsApp/FB 显示的图）
└── dist/tailwind.css       网站样式（自动生成，不要手改）
```

> ⚠️ `dist/tailwind.css` 是自动构建的。如果改了 HTML 里的样式类(class)，需要重新构建（`npm run build:css`）——交给 Claude 最省心。

## 7. 日常运营建议

- **每天**：及时回复 WhatsApp 询盘（网站所有按钮都导向 WhatsApp，这是你的命脉）。
- **每周**：看一眼 GA——这周多少访客、哪些国家、有几个 `generate_lead`（咨询）。
- **每月**：看 GSC 效果报告——你在 Google 上靠什么词被搜到、排名有没有上升。
- **持续**：
  - 多上架机器（每台 = 一个新页面 = 更多搜索入口）
  - 收集客户好评，让 Claude 加到网站（外贸信任神器）
  - 让 Claude 定期写买家指南文章（吸引搜索流量）
