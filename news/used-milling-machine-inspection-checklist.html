Used Milling Machine Inspection Checklist

Before buying a used milling machine from China, buyers should confirm more than the model name and price. A good inspection helps reduce repair risk, shipping disputes, and unexpected downtime after the machine arrives.

1. Confirm running condition

Ask for a clear video of the machine running. The video should show spindle start, speed change, table movement, power feed, and basic cutting or test movement when possible.

2. Check key parts

  • Spindle noise and vibration
  • Table surface, T-slots, and travel smoothness
  • Gearbox sound and speed changes
  • Electrical cabinet and control buttons
  • Oil leakage around the head, knee, and feed box

3. Request photos before payment

Useful photos include the nameplate, full machine view, table close-up, control panel, motor, accessories, and any repaired or repainted areas.

4. Confirm export preparation

Before shipment, confirm cleaning, rust protection, wooden base or pallet, loading photos, and container seal photos. These records make the export process easier to verify.
